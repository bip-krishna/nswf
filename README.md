# NSWF — Nisarg Srishti Welfare Foundation

Official website for **Nisarg Srishti Welfare Foundation (NSWF)**, a Pune-based NGO working on noise pollution awareness, environmental education, and community empowerment since 2023.

## About

NSWF builds a quieter, healthier, and more sustainable India through awareness campaigns, community action, and evidence-based environmental advocacy. This site serves as the organisation's digital presence — communicating its mission, showcasing programmes and impact, and converting visitors into supporters.

## Pages

- **Home** — Mission, impact metrics, featured programmes, and calls to action
- **About** — Organisation story, vision, and founder Dr. Garima Kavathekar
- **Programs** — Active initiatives and campaigns
- **Events** — Past and upcoming events
- **Impact** — Measurable outcomes and success stories
- **Media** — Press coverage and media resources
- **Team** — Leadership and members
- **Ambassadors** — Brand ambassadors
- **Advisors** — Advisory board
- **Policies** — Organisation policies
- **Membership** — How to join and support
- **Projects** — Ongoing and completed projects
- **Contact** — Get in touch

## Tech Stack

- Static HTML5 with semantic markup
- CSS (custom design system — dark editorial aesthetic)
- Vanilla JavaScript
- WCAG AA compliant, reduced-motion supported

## Setup

```bash
git clone https://github.com/bip-krishna/nswf.git
cd nswf
# Open index.html in a browser or serve locally:
python3 -m http.server 8000
```

## Design

The site follows a cinematic, dark editorial aesthetic with generous whitespace. Impact evidence, real photography, and clear single-action pages drive the experience. See [`PRODUCT.md`](./PRODUCT.md) for the full design rationale.
